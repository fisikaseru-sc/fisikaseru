Template laporan memuat:

- Identitas percobaan + tujuan.
- Metode dan setelan (`d`, `V_obj`, `T`, `P`, `eta`, mode Cunningham).
- Ringkasan tabel data.
- Grafik wajib.
- `e_est` dan perbandingan terhadap `1.6 x 10^-19 C`.
- CER + pembahasan sumber error:
  - reaksi timer,
  - pemilihan droplet,
  - ketidakpastian pembacaan `U`,
  - jitter crossing mark.

Gunakan tombol export untuk menghasilkan JSON/CSV atau PDF via print pipeline browser.
